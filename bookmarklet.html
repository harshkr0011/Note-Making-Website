Nexus Notes Web Clipper

How to Install

Method 1: Drag this button to your bookmarks bar:

Clip to Nexus Notes

Method 2: Copy and paste this code into a new bookmark:

javascript:(function(){var script=document.createElement('script');script.src='http://localhost/Nexus%20Notes/assets/js/web_clipper.js?'+new Date().getTime();document.body.appendChild(script);})();
Copied!

How to Use

  1. Navigate to any webpage you want to save
  2. Optional: Select specific text you want to save
  3. Click the "Clip to Nexus Notes" button in your bookmarks bar
  4. Review and edit the content in the popup window
  5. Add tags and select a folder (optional)
  6. Click "Save" to store it in your Nexus Notes

Features

  • Clip selected text or entire pages
  • Automatically captures page title and URL
  • Organize clips with folders
  • Add tags for better organization
  • Works on any webpage
  • Saves directly to your Nexus Notes account

Important Notes

  • Make sure you're logged into Nexus Notes before using the Web Clipper
  • The Web Clipper works best with modern browsers (Chrome, Firefox, Safari, Edge)
  • Some websites may block content selection - in these cases, the clipper will try to capture the main content
